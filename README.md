# RNDR Price Tracker

Simple Python script that tracks RNDR price using the CoinGecko API.

This tool prints the current RNDR price in USD and can be used to watch for moves above or below selected levels.

Features

- Pulls live RNDR price data from CoinGecko
- Displays current price in USD
- Runs continuously with adjustable refresh interval
- Alerts when price moves above or below selected levels

Setup

Install dependencies:

pip install -r requirements.txt

Run the script:

python rndr_price_tracker.py
